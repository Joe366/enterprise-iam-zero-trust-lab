# Enterprise IAM Zero Trust Lab

## Overview
This project demonstrates the design and implementation of an enterprise Identity and Access Management (IAM) architecture using Microsoft Entra ID, Microsoft Intune, and Microsoft 365 security capabilities.

The lab simulates a real-world organization with multiple departments and implements Zero Trust security principles including Conditional Access, Privileged Identity Management, device compliance enforcement, and identity governance.

---

## Project Objectives

- Implement department-based RBAC
- Secure privileged access using PIM
- Implement Zero Trust Conditional Access policies
- Require device compliance for administrative access
- Automate identity governance with Access Packages
- Monitor identity risk and sign-in activity
