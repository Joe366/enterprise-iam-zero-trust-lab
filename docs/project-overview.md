# Project Overview

This project simulates a mid-size organization implementing a secure Identity and Access Management architecture using Microsoft Entra ID and Microsoft 365 security tools.

The environment contains several departments:

- Information Technology
- Finance
- Human Resources
- Clinical Operations

Each department has its own security group and access model.

The project focuses on implementing Zero Trust identity security where every authentication request must be validated based on identity, device health, and security policies.

Security features implemented include:

- Conditional Access enforcement
- Privileged Identity Management
- Device compliance with Intune
- Identity governance with Access Packages
- Risk-based sign-in monitoring
