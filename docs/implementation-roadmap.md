# Implementation Roadmap

Phase 1 — Identity Structure  
Users and departments created.

Phase 2 — Security Groups  
Department-based RBAC groups created.

Phase 3 — Group Membership  
Users assigned to department groups.

Phase 4 — Role Assignments  
Administrative roles assigned to IT staff.

Phase 5 — Privileged Identity Management  
Admin roles configured as eligible with MFA and approval requirements.

Phase 6 — Conditional Access  
Zero Trust Conditional Access policies implemented.

Phase 7 — Device Compliance  
Intune compliance policies created.

Phase 8 — Identity Governance  
Access Packages created for departments.

Phase 9 — Future Work  
Automated provisioning with Microsoft Graph.
